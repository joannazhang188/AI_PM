# 模板-产品功能更新说明线-v1

- 文档类型：模板
- 适用阶段：产品功能更新说明线
- 用途：根据 PRD 或版本内容生成对内或对外的产品功能更新说明
- 触发条件：版本内容基本明确，用户要求输出更新说明
- 上一阶段输入：PRD / 版本范围 / 实际交付内容
- 下一阶段输出：飞书文档 / 发布说明 / 对内同步材料
- 版本：v1.0

## 字段填写说明

- 更新背景：写为什么有这次更新。
- 更新范围：写本次实际涉及的模块和能力。
- 关键变化：写新增、优化、调整，不要只写“若干优化”。
- 用户影响：明确谁会感知到变化。
- 注意事项：写限制、兼容性、已知问题和使用建议。
- 不写内容：避免承诺尚未上线的功能。
